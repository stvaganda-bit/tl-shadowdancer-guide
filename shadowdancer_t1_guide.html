Обзор

ShadowDancer — архетип Spear + Dagger в Throne and Liberty. В T1 PvP занимает A-Tier как в Large Scale, так и в Arena (Maxroll, март 2026). Класс требует высокого skill ceiling, особенно в Arena, но при правильной игре обеспечивает одни из лучших AoE-бёрстов в Large Scale и мощные сингл-таргет ассасин-комбо в Arena.

Важно: ShadowDancer ≠ Darkblighter ShadowDancer = Spear + Dagger. Darkblighter = Wand + Dagger. Это разные классы. Не путай билды.

Что такое ShadowDancer

ShadowDancer — мобильный дайвер и бёрст-ассасин, использующий копьё для дальних AoE-атак и кинжал для ближних CC-цепочек. Отличительная черта класса — Shadow Walker: пассивное уклонение, которое активируется при каждом использовании мобильного скилла. Это означает, что класс должен находиться в постоянном движении.

Javelin Inferno — главный козырь в Large Scale: рикошетит между сгруппированными врагами, нанося массовый урон. В Arena — скрытный ассасин с жёсткой CC-цепочкой.

Роль в T1

Large Scale
  • Backline diver — ныряет в тыл к хилерам и рейнджам противника
  • AoE bomb dealer — Javelin Inferno рикошетит между врагами = максимальный потенциал AoE
  • Lightning Infusion + Thunder Spirit одновременно перед Javelin = экспоненциальный буст
  • Camouflage Cloak — инструмент репозиционирования ДО атаки, не побег
  • Phantom Smokescreen — блокирует AoE и Chain Hook во время дайва
Arena
  • Single-target burst assassin — заход из стелса
  • Vicious Fury → Shadow Strike (бинд + сайленс) — стандартный опенер
  • CC-цепочка: бинд → стан → прон = окно для бёрста
  • Один пропущенный шаг = весь комбо рушится

Ключевые ограничения

Ammo Management Javelin Inferno требует 5 Embers (накапливаются через Explosive Fury за 30 секунд на глобальном сервере). Без 5 Embers — урон минимален. Это главное ограничение класса.
Combo Dependency (Arena) Любой пропуск в CC-цепочке (бинд → стан → прон) обнуляет весь бёрст-потенциал. Требует стабильного пинга и практики до автоматизма.
Антидот для слабостей Shadow Walker активен только при постоянном использовании мобильных скиллов. Нет движения = нет уклонения = прямой урон в лицо. CDR на мобильные скиллы — всегда приоритет.

Роль в Large Scale

Backline diver + AoE bomb dealer. Задача — нырнуть в тыл к хилерам и рейнджам противника, взорвать Javelin Inferno в кластере, уйти живым.

Золотое правило Large Scale Принимай расчётные смерти ради успешного bombing run. Kill streaks восстанавливают Embers для следующего Javelin.

Советы Large Scale

Тайминг Lightning Infusion + Thunder Spirit Оба баффа должны быть активны одновременно перед броском Javelin. Раздельное использование даёт вдвое меньший эффект.
Phantom Smokescreen — проактивно Используй при попытке Chain Hook или во время mass AoE по твоей позиции. Не жди пока прилетит.
Shadow Walker — движение = жизнь Каждый мобильный скилл активирует уклонение. Стоять = умирать. CDR на мобильные скиллы обязателен.
Не кидай Javelin в одну цель Механика рикошета работает ТОЛЬКО при наличии нескольких врагов рядом. Одиночный бросок — слабый урон.

Роль в Arena

Single-target burst assassin. Заход из стелса, жёсткая CC-цепочка, максимальный урон в узком окне, пока цель в CC. Один пропущенный шаг = весь комбо рушится.

Не используй LS AoE ротацию в Arena Javelin Inferno / Explosive Fury / AoE комбо не работают 1v1. В любом small-scale бою переключайся на CC chain ротацию.

Топ-8 распространённых ошибок

Ошибка #1
Javelin Inferno без 5 Embers
Запуск без полного стека = минимальный урон. На Global — 30 секунд на накопление 5 Embers. Не торопись.
Ошибка #2
Javelin Inferno в одну цель
Механика рикошета работает ТОЛЬКО при наличии сгруппированных врагов. Одиночный бросок = слабый урон. Позиционируйся перед броском.
Ошибка #3
Lightning Infusion и Thunder Spirit по отдельности
Оба баффа нужны ОДНОВРЕМЕННО перед Javelin. По отдельности = половина эффекта.
Ошибка #4
AoE ротация в Arena
LS комбо не работает 1v1. Переключайся на CC chain для любого small-scale боя.
Ошибка #5
Camouflage Cloak как реактивный побег
Cloak — инструмент репозиционирования ДО атаки. Реактивное использование = потеря бёрст-окна.
Ошибка #6
Пропуск шага CC-цепочки в Arena
Весь комбо рушится. Критическая потеря DPS. Тренируй до автоматизма.
Ошибка #7
Нет мобильности для Shadow Walker
Отсутствие нажатий мобильных скиллов = нет пассивного уклонения = урон в лицо. Shadow Walker активен только при постоянном движении.
Ошибка #8
Guardians в панике
Hartach и Kamarshea ставятся проактивно в начале боя. Реактивное использование = потеря эффекта. Hartach активируется только на убийствах — нужна активная фаза боя.
