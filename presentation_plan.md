## Формат блоков

### Таблицы
- Core Build навыки (оба режима)
- Stats Priority (STR/DEX/PER по этапам)
- Equipment progression
- Matchups
- Sources

### Карточки
- Сильные/слабые стороны
- Навыки Core Build (иконка + название + описание)
- Flex Build Variations (замены, сценарий, плюсы, минусы)
- Common Mistakes

### Ротации
- Визуализация: Skill → Skill → Skill (pill-стиль)
- Пояснения: цель / когда / результат / ошибки
- 6 ротаций всего (3 LS + 3 Arena)

### Схемы позиционирования
- Large Scale: До боя → Вход → Во время → Выход
- Arena: Дистанция → Stealth → CC lock → Finish

### Cheat Sheet
- Отдельный fullscreen экран
- Core Build (список)
- Основная ротация (визуальная цепочка)
- Приоритет целей
- Топ советы
- Топ ошибки
